# C-Programs

A collection of C programming examples and utilities.

## Contents

- **HulaireCalculator.c** - A calculator program that performs basic arithmetic operations

## Getting Started

### Prerequisites

- GCC compiler or any C compiler
- Terminal/Command prompt

### Compilation

```bash
gcc HulaireCalculator.c -o calculator
```

### Running

```bash
./calculator
```

## Usage

The calculator allows you to perform basic arithmetic operations. Follow the on-screen prompts to:
1. Select an operation
2. Enter your numbers
3. View the result
